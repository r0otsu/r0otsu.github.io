Task 0. 在CIFAR-10上实现LeNet-5模型

实验步骤

  1. 首先进行基础设置和超参数的设定，定义了每次训练的图片BATCH_SIZE=64、学习率LEARNING_RATE=0.001、训练轮次EPOCHS=30、以及自动检测是否用gpu

    初次尝试时，在10epoch下，无法达到65%的acc，大概是60-65%，所以这里优先选择了增加训练轮次到30次

    task0因为开销还没有特别大，就用本机跑的，是cpu

  2. 初次尝试时，在10epoch和30epoch下，LeNet都无法达到65%的acc，大概是60-65%，所以需要在不改变模型结构的条件下，另寻其他的方法增强模型的分类能力

    这里选择了数据增强的方法，在训练集中加入了随机翻转和裁剪，模型在每一轮看到的图片都略有不同，有所翻转或裁剪，使得模型学习物体本质特征，而不是直接记忆像素点，能够有效防止过拟合，达到数据增强的目的

  3. 这里根据前置知识中的LeNet-5模型的C1-S2-C3-S4-C5-F6进行建模

    模型结构代码如下：

    模型具体维度如下：

    • 输入图片：32×32
    • self.conv1 = nn.Conv2d(3, 6, 5) -> 3表示(R,G,B)三通道，6表示输出6个特征，5表示卷积核的大小是5×5 -> Conv1(5×5卷积): (32-4)×(32-4)=28×28
    • self.pool = nn.MaxPool2d(2, 2) -> 把图片的长宽缩小一半 -> Pool1(2×2池化): 14×14
    • self.conv2 = nn.Conv2d(6, 16, 5) -> 6表示C1输出的6个通道，16表示增加到16个通道，5表示卷积核的大小是5×5 -> Conv2(5×5卷积): (14-4)×(14-4)=10×10 通道数此时是16
    • self.fc1 = nn.Linear(16 * 5 * 5, 120) -> C2后展平后的向量长度是 16×5×5=400，120表示输出的神经元数量
    • self.fc2 = nn.Linear(120, 84) -> fc1后展平为120，输出84
    • self.fc3 = nn.Linear(84, 10) -> fc2后展平为84，输出10，而10正是10个分类的类别

  4. 前向传播

    前向传播函数输入x是指输入的图片数据

    • x = self.pool(F.relu(self.conv1(x)))

      卷积 -> 激活(ReLU) -> 池化

      32×32 -> 28×28 -> 14×14

    • x = self.pool(F.relu(self.conv2(x)))

      卷积 -> 激活(ReLU) -> 池化

      14×14 -> 10×10 -> 5×5

    • x = x.view(-1, 16 * 5 * 5)

      展平 (Flatten)

      view(-1, …)能够自动计算批次大小，把立体的 (Batch, 16, 5, 5) 变成平铺的 (Batch, 400)

    • x = F.relu(self.fc1(x))

      全连接层

      400 -> 120

    • x = F.relu(self.fc2(x))

      全连接层

      120 -> 84

    • x = self.fc3(x)

      全连接层

      84 -> 10 返回 10 个分类各自的得分

  5. 训练阶段，通过循环计算loss并反向传播更新参数

    为了防止过拟合，设置了权重衰减：weight_decay=1e-4，限制权重参数不要太大，让模型更平滑

  6. 验证阶段

    在每个epoch结束后，计算一遍测试集的acc，不仅为了防止过拟合，也是为了确认训练的效果

  7. 存储模型

  8. 最后通过保留的最佳参数模型，再验证一遍测试集

实验结果

训练过程与测试结果输出，超过了65%

实验结果表明，通过数据增强和权重衰减，能够在一定程度上改进模型性能。

Task 1. 将预训练深度卷积模型在CIFAR-10上做迁移

实验环境：AutoDL+Jupyter Lab

模型选择：ResNet50

ResNet50是ResNet系列中的一个变种，包含50个卷积层。

其整体架构可分为三部分：

  • 头部(Head)：用于初始特征提取
  • 主体(Body)：多个残差块堆叠
  • 尾部(Tail)：分类器

ResNet50的层次结构如下：

  • 7×7卷积层，步长为2
  • 3×3最大池化层，步长为2
  • 4个残差块组，每组包含多个Bottleneck残差块
  • 全局平均池化
  • 全连接层

实验步骤

  1. 首先也是基础设置和超参数定义

  2. 数据预处理

    初始化：使用 ImageNet 的统计数据（0.485…），因为预训练模型的权重是基于这个分布训练的

    Resize((224, 224))：放大图片，看清细节，便于分类

    RandomHorizontalFlip()：和task0一样随机翻转，进行数据增强

  3. 读取数据

  4. 从torchvision上下载预训练的ResNet50，直接使用 PyTorch 官方提供的ResNet50训练好的最新、最强权重

  5. 冻结神经网络层

    这里因为我之前试过只冻结fc，加训练轮次也无法超过90%，只能到85%上下，所以选择了再解冻一层

    而ResNet的最后一层卷积块提取的是高级语义特征，ImageNet的类别和 CIFAR-10差异较大，所以使用预训练的ResNet完全不微调，性能上非常受限，因此再多解冻一层

  6. 改进分类头

    ResNet50输出的特征向量很大，包含极大信息量，简单的单层线性层 (Linear(2048, 10))能力有限，容易造成“瓶颈”

    通过引入一个隐藏层（MLP结构），能更稳健地将高维特征映射到分类结果上

  7. 优化器和学习率调度

    显式告诉优化器只更新解冻的层，节省内存

    并且学习率不断衰减，防止过拟合

  8. 训练与测试

    同样地，通过前向传播、反向传播的标准流程，并且在每个epoch结束后进行测试

    不过需要每一个epoch都更新学习率
